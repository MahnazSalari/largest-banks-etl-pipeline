# Largest Banks ETL Pipeline

A production-ready Python ETL pipeline that automates the extraction, transformation, and loading of global banking data.

## Project Overview
This project extracts data on the world's largest banks by market capitalization from Wikipedia, transforms the data by converting currency metrics based on exchange rates, and loads the processed results into both a CSV file and a localized SQLite database for analytical queries.

## Tech Stack
* **Language:** Python
* **Libraries:** BeautifulSoup, Pandas, SQLite3
* **Automation:** Logging integration for pipeline monitoring
